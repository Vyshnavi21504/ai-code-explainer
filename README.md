# 🤖 AI Code Explainer

A high-performance, full-stack application that translates complex code into plain English. This project leverages **FastAPI** for a robust, secure backend and **Streamlit** for an intuitive, interactive frontend.

---

## 📺 Project Walkthrough


---

## ✨ Key Features
* **Deep Logic Analysis:** Breaks down nested loops, recursions, and complex data structures.
* **Secure API Handling:** API keys are managed server-side via FastAPI to prevent client-side exposure.
* **Multi-Language Support:** Explain snippets from Python, JavaScript, C++, Rust, and more.
* **Real-time Response:** Fast, asynchronous processing for a smooth user experience.

---

## 🛠️ Tech Stack

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | [Streamlit](https://streamlit.io/) | User Interface & Input Handling |
| **Backend** | [FastAPI](https://fastapi.tiangolo.com/) | API Logic & Key Management |
| **AI Engine** |  Gemini | Code Interpretation |
| **Communication** | REST API | JSON Data Exchange |
